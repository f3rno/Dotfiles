# Installation
```sh
$ git clone https://github.com/Alvarocz/vim-northpole.git && cd vim-northpole
```

## Using vim
```sh
$ cp colors/northpole.vim ~/.colors/ (or whatever directory you use)
```

## Using neovim
```sh
$ cp colors/northpole.vim ~/.config/nvim/colors/
```
